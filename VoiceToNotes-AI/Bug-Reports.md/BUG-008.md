## Environment

* **Operating System:** Ubuntu 24.04 LTS
* **Primary Browser:** Google Chrome (Latest Version)
* **Secondary Browsers Tested:** Mozilla Firefox (Latest Version)
* **Platform:** Desktop
* **Testing Type:** Manual Functional Testing
* **Build/Test Date:** During QA Technical Assessment

## Severity: **Major**

## Priority: **P1 – High**


## Steps to Reproduce

1. Open the VoiceToNotes.ai website using a browser other than Google Chrome (e.g., Mozilla Firefox).
2. Navigate through the landing page and available application features.
3. Compare the application's behavior with the experience in Google Chrome.
4. Observe any compatibility, rendering, or functional issues.

## Expected Result

The application should provide a consistent and functional user experience across all major modern browsers, including Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.

## Actual Result

The application exhibits compatibility issues when accessed through browsers other than Google Chrome. 

Certain pages, UI components, and features do not function or render as expected, resulting in an inconsistent user experience across supported browsers.

## Conclusion

The application currently demonstrates browser compatibility issues outside Google Chrome, which may prevent users of other modern browsers from accessing all features reliably. 

Improving cross-browser compatibility would enhance accessibility, usability, and overall application reliability.
