# Bug Reports

This directory contains the bug reports identified during the **VoiceToNotes.ai QA Technical Assessment**. 

Each defect was reproduced through manual functional testing and documented using the reporting format specified in the assessment guidelines.

Every report includes the affected module, testing environment, severity, priority, reproduction steps, expected and actual behavior, along with supporting evidence where applicable. 

The objective of these reports is to provide clear, reproducible information that assists the development team in understanding and resolving the identified defects efficiently.

- **BUG-001** – Mobile Features Page Displays "Not Supported" Message
- **BUG-002** – Test Login Credentials Missing from Assessment
- **BUG-003** – Voice Recording Does Not Generate Transcription After Microphone Permission
- **BUG-004** – Mobile Landing Page Displays Error Message
- **BUG-005** – Voice Transcription Experiences Significant Delay
- **BUG-006** – UI/UX Rendering Issues (Overlapping Elements and Poor Readability)
- **BUG-007** – Images Load Slowly or Require Manual Page Refresh
- **BUG-008** – Excessive Unused JavaScript Affecting Performance
