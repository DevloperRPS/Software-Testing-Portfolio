# BUG-001 : Employee Profile Picture Not Displayed in Directory Module

During manual testing of the OrangeHRM application, an issue was observed in the **Directory** module where employee profile pictures failed to load while the remaining employee information was displayed correctly. 

The issue was documented for further investigation.


## Observation

While viewing employee profiles in the Directory module, the profile picture was not displayed even though other employee details were loaded successfully.

## Impact

The issue does not interrupt application functionality but affects the visual presentation of employee profiles and may reduce the overall user experience.

## Recommendation

- Verify that employee profile images are correctly retrieved and rendered.
- Ensure image resources are available and accessible.
- Display a default placeholder image if a profile picture is unavailable.

## Conclusion

The issue affects the visual consistency of the Directory module and should be investigated to improve the overall usability and presentation of employee information.
