# OrangeHRM Manual Testing Project

Welcome to the OrangeHRM Software Testing Project.

This repository showcases the manual testing activities performed on the OrangeHRM demo application. The project demonstrates practical software testing techniques, documentation standards, and quality assurance practices using real-world testing scenarios.

---

## Project Objective

The primary objective of this project is to validate the core functionalities of the OrangeHRM application through systematic manual testing. The project also aims to strengthen QA documentation skills by preparing professional test cases, execution summaries, and automated quality assessment reports.

---

## Application Overview

OrangeHRM is an open-source Human Resource Management (HRM) system used by organizations to manage employee records, recruitment, leave management, performance tracking, and administrative operations. Due to its rich functionality, it serves as an excellent platform for practicing software testing techniques.

---

## Modules Covered

The testing activities in this project include the following modules:

- Login & Authentication
