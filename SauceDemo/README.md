Welcome to my **SauceDemo** Manual Testing Project.

This Project is a part of my Software Testing Portfolio, where I perform end-to-end manual testing on real world and demo applications. Here I have performed Manual Testing on SauceDemo Web-Application.

## About SauceDemo
**SauceDemo** is a demo e-commerce web application widely used by QA Professionals and learners for practicing software testing. It provides various modules such as authentication, product browsing, shopping cart, and checkout, making it an ideal platform to perform functional testing.

## Primary Objective
The primary objective is to evaluate the application's functionallity from an end-user perspective. Every feature is tested systematically to verify that it behaves according to the expected requirements while documenting any issues or observations found during testing.

## Scope of Testing
The following modules are covered throughout this testing project.

- Login & Authentication
- Product Listing
- Product Details
- Shopping Cart
- Checkout Process
- Navigation Menu
- Logout Functionality
- Form Validation
- UI/UX Verification
- Functional Testing
